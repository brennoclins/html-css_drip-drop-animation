#### Como criar gif animada com imagens das telas de suas aplicações.

- No linux abra o termina e digite o comando abaixo:
```bach
convert -delay 120 -loop 0 *.png criandoGIFcomImagens_by_Brenno_BCL-LAB.gif
```
